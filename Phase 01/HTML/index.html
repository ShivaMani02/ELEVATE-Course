This page is a comprehensive guide to the most important HTML tags, their usage, and examples. HTML (HyperText Markup Language) is the standard language for creating web pages and web applications.

<h1> to <h6>: Heading Tags

Headings are used to define the structure of your content.
<h1> is the most important (largest), and <h6> is the least important (smallest).

<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection</h3>

<p>: Paragraph Tag

The <p> tag defines a paragraph of text. Browsers automatically add some space before and after each paragraph.

<p>This is a paragraph of text.</p>

<a>: Anchor Tag

The <a> tag creates hyperlinks, which allow users to click and navigate to another page or website.
The href attribute specifies the destination URL.

<a href="https://www.example.com">Visit Example</a>

<img>: Image Tag

The <img> tag is used to display images.
src specifies the image URL, and alt provides alternative text for accessibility.

<img src="https://via.placeholder.com/150" alt="Sample Image">

<ul>, <ol>, <li>: Lists

Lists are used to group related items:

  • <ul>: Unordered (bulleted) list
  • <ol>: Ordered (numbered) list
  • <li>: List item

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
<ol>
  <li>First</li>
  <li>Second</li>
</ol>

<table>: Table Tag

Tables organize data into rows and columns. Use <tr> for rows, <th> for headers, and <td> for data cells.

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>
Header 1 Header 2
Data 1 Data 2

<form>: Form Tag

Forms collect user input. Common elements include <input>, <label>, <textarea>, <select>, and <button>.

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>







<div> and <span>: Structural Tags

<div> is a block-level container for grouping elements, useful for layout.
<span> is an inline container for styling or marking up a part of text.

<div style="background: #eee;">Block content</div>
<p>This is <span style="color: red;">important</span> text.</p>

<input>: Input Tag

The <input> tag is used for various types of user input, such as text, password, checkbox, radio, etc.

<input type="text" placeholder="Text">
<input type="checkbox"> Checkbox
<input type="radio" name="choice"> Radio

<footer>: Footer Tag

The <footer> tag defines a footer for a document or section, often containing copyright or contact info.
